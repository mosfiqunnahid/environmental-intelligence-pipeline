# Environmental Intelligence Pipeline

An end-to-end environmental data engineering and geospatial analytics platform designed for real-time urban climate monitoring using REST APIs, automated ETL pipelines, environmental data processing, GIS visualization, and interactive dashboards.

## Key Features
- Real-time environmental data ingestion using REST APIs
- Automated ETL workflow for climate data processing
- JSON transformation and structured dataset generation
- Exploratory environmental data analysis (EDA)
- GIS-based geospatial visualization using Folium
- Interactive dashboard development with Streamlit

## Technologies
Python • REST APIs • Pandas • Streamlit • Folium • Matplotlib • GIS • ETL

## Workflow

```text
API → ETL Pipeline → Structured Dataset → Environmental Analytics → GIS Visualization → Dashboard
```

## Run

```bash
python3 src/main.py
streamlit run dashboard/app.py
```
