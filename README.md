# Environmental Intelligence Pipeline

A climate and environmental data analytics system built using real-time weather APIs, ETL workflows, GIS visualization, and interactive dashboards.

## Features
- Collect real-time weather data from APIs
- Process and clean environmental data
- Analyze climate data using EDA
- Create GIS-based map visualization
- Build interactive dashboards

## Technologies
Python • REST APIs • Pandas • Streamlit • Folium • GIS

## Workflow

```text
API → Data Processing → Analysis → GIS Map → Dashboard
```

## Run

```bash
python3 src/main.py
streamlit run dashboard/app.py
```
